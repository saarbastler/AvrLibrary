Detailed Description

template<uint8_t FIFO_SIZE, uint8_t I2C_ADDRESS, bool DISPLAY_LINES = false, bool FONT = false>
class SABA::I2C::LcdText< FIFO_SIZE, I2C_ADDRESS, DISPLAY_LINES, FONT >

A template class to access a I2C Text LCD.

Template Parameters
FIFO_SIZEthe putch Fifo size for non blocking printing
I2C_ADDRESSthe I2C Device Adress, tpyicalle 0x4e
DIPLAY_LINESfalse for 1 line, false for 2 lines (see LCD Datasheet)
FONTfalse = 5*7 font
