Detailed Description

template<SFRA PIN_ADDR>
union SABA::Port8< PIN_ADDR >

The union will reduce the size to one byte instead of three bytes. Even an empty C++ class will consume one byte. Do not use this union to access single port pins, use the PortPin class instead.

Template Parameters
PIN_ADDRthe PINx address, DDRx and PORTx is calculated

Usage:

SABA::Port8<(SFRA)PINC> pc;
pc.ddr= 0xff; // all outputs
pc.port= 0x00; // all zero
