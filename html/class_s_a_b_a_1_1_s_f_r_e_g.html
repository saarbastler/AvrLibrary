Detailed Description

template<SFRA SFR_ADDRESS>
class SABA::SFREG< SFR_ADDRESS >

A template class to access AVR special function registers. The C++ compiler will compile the methods to single assembler instructions.

Template Parameters
SFR_ADDRESSthe SFR address

Usage:

SABA::SFREG<(SFRA)&TCNT0> tcnt0;
uint8_t value= tcnt0(); // read the TCNT0 register
tcnt0= 0x12; // write the TCNT0 register
